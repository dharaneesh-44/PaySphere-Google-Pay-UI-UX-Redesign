# 💳 GPay Mobile App UI Design

A modern **Google Pay (GPay) inspired mobile payment UI** designed in **Figma**. This project showcases a clean, intuitive, and user-friendly digital payment experience with both **Light Mode** and **Dark Mode** interfaces, focusing on seamless money transfers and everyday payment features.

---

## ✨ Project Overview

This UI design recreates a modern digital payment experience inspired by popular UPI payment applications. The project includes multiple screens covering the complete payment flow—from the home dashboard to chat-based transactions and payment confirmation.

Designed with simplicity, accessibility, and consistency in mind, the interface provides a smooth user experience for mobile payment applications.
